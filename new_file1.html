一、如果文件夹或者项目没有git仓库，那么需要新建一个仓库


(1)终端进入该文件夹，然后输入git init指令

(2) 创建完毕之后，此时你是处于master分支下的

(3)克隆项目仓库 git clone 仓库链接


二、如果拿到了项目文件夹，此时作为小组成员，你需要新建一个分支



(1) 创建分支 git checkout -b 分支名称

(2) 克隆项目仓库 git clone 仓库链接

(3)将本文件夹内的所有数据列表进行刷新 git add .

(4) git commit -m "备注信息" 正式提交代码 //这是提交到本地仓库

(5)如果要把本地仓库的代码，推送到线上仓库: git push -u origin 线上仓库的分支的名称

三、分支的基本操作


(1)切换分支

git checkout 分支名称

(2) 合并分支，其他分支的代码合并到当前的分支下

git merge 分支名称

(3)更新线上分支

git pull origin 你要更新的分支的名称
